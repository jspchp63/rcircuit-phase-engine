# RCIRCUIT v1.1 — Phase Compute Primitive (Minimal Skeleton)

## Purpose
Define the simplest possible compute primitive for RCIRCUIT:
**phase update + Δsignal + stability check.**

---

## 1. Primitive Components
- `phase` — node’s internal oscillation state  
- `delta_signal` — meaningful change only (no full value movement)  
- `modulation` — small constant controlling update magnitude  
- `stability_flag` — STABLE / UNSTABLE  
- `resonance_score` — long-term coherence indicator  

---

## 2. Primitive Rule (Core)
phase_new = phase_old + delta_signal * modulation


Local-only.  
No global memory shuffle.  
Movement-minimizing compute.

---

## 3. Stability Check
delta_phase = abs(phase_new - phase_old)

if delta_phase < STABILITY_THRESHOLD:
stability_flag = STABLE
else:
stability_flag = UNSTABLE
