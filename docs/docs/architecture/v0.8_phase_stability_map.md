# RCIRCUIT v0.8 — Phase Stability Map (Skeleton)

Stability is achieved when local phase variance stays below a coherence threshold ε.
Noise amplifies when ∂phase/∂t exceeds the coherence window.
Stability window W = [φ - ε, φ + ε].
Collapse risk = variance / ε.
